LeetCode 2 两数相加

给定两个非空链表来表示两个非负整数。位数按照逆序方式存储，它们的每个节点只存储单个数字。将两数相加返回一个新的链表。

你可以假设除了数字 0 之外，这两个数字都不会以零开头。

示例：

输入：(2 -> 4 -> 3) + (5 -> 6 -> 4)
输出：7 -> 0 -> 8
原因：342 + 465 = 807

C++ new：

  • 在堆上分配空间
  • 在分配的空间上调用对象的构造函数

deleta:

  • 首先调用 这个对象 的析构函数
  • 然后释放这个对象的空间

这题的关键在于

创建一个指向ListNode的指针，并让它指向一个val为0的ListNode的时候，你要为它new一个空间出来不然会报错：
Line 20: member access within null pointer of type ‘struct ListNode’
