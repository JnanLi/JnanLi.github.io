数字图像处理

2018-07-03

参考书籍：《数字图像处理与分析》张弘、《数字图像处理的MATLAB实现（第2版）》RCG

1. 绪论

模拟图像是通过某种物理量（光、电等）的强弱变化来记录图像上各点的亮度信息的；数字图像是用数字（即计算机存储的数据）来记录图像各点的亮度信息。数字图像是由模拟图像数字化或离散化得到的。

当x、y分量及幅值f都是有限且离散的量时，我们称图像为数字图像。

关于像素和灰度：像素是数字图像的基本元素，每个像素具有整数行和列位置坐标；灰度表示像素所在位置的亮度，彩色图像一般采用红、绿、蓝三基色的颜色值。

2. 图像处理基础知识

3. MATLAB基础

3.1 数字图像的表示

首先，工具箱用(r,c)而不是(x,y)来表示行和列，这个坐标系统的原点在(r,c)=(1,1)处。因此，r是从1到M的整数，c是从1
到N的整数。

3.2 图像的输入/输出和显示

可以使用函数imread将图像读入MATLAB环境。

4. 图像增强

图像增强的目的有二，一是改善图像的视觉效果，提高图像的清晰度；二是将图像转换成一种更适合于人类或机器进行分析处理的形式，以便图像中获取更多有用的信息。

图像增强算法的应用是有针对性的，并不存在通用的增强算法。

说一下彩色增强，对于灰度图像，人眼能分辨的灰度级只有十几到二十几，而对不同亮度和色调的彩色图像则能达到几百甚至上千。利用这一特性，将颜色信息用于图像增强之中能提高图像的可分辨性。
伪彩色增强是针对灰度图像提出的，其目的是吧离散灰度图像f(x,y)的不同灰度级按照线性或者非线性关系映射成不同的彩色。
真彩色增强所处理的对象不是一副灰度图像，而是一副自然彩色图像或是同一景物的多光谱图像，它是一种彩色到彩色的变换。其目的一是变换图像的彩色，引起人们的特别注意，而是根据人眼对不同颜色的灵敏度不同，使景物呈现出与人眼视觉相匹配的颜色。
