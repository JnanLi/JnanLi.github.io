Basic C++ Programming

1.如何撰写C++程序

函数是一块独立的程序代码序列，包括返回值类型、函数名称、参数列表以及函数体。

main()函数返回整数类型，习惯上，程序执行无误时我们令main()返回0，若返回一个非0值，表示程序在执行过程中发生了错误。

C++的基础数据类型包括：布尔值(Boolean)、字符(character)、整数(integer)、浮点数(floating point)。

数据的输入与输出，并非C++程序语言本身定义的一部分(此精神tongC语言)，而是由C++的一套面向对象的类层次体系提供支持。欲使用class，我们必须先在程序中包含其头文件。头文件可以让程序知道class的定义。C++标准的”输入/输出库”名为iostream，用以支持对 终端 和 文件的输入与输出。

using和namespace都是C++里的关键字。std是标准库所驻之命名空间(namespace)的名称。标准库所提供的任何事物(诸如string class以及cout、cin这两个iostream类对象)都被封装在命名空间std内。using namespace是让命名空间曝光的最简单办法。

命名空间是一种将库名称(诸如iostream、string)封装起来的方法。通过这种方法，可以避免和应用程序发生命名冲突的问题。
